# EPIC-003: Time-Horizon Views
# Agent: 02 — PRODUCT OWNER | Standards: coding-standard v1.0 | 2026-06-19T12:30:00Z

## Summary
Authenticated users can view distinct dashboards filtered by timeframe (Daily, Weekly, Monthly), enabling focused attention on the appropriate planning horizon.

## Business Value
Differentiation feature — separates TaskFlow from generic to-do apps. Addresses Jamie's (PhD student) core pain: overwhelming mix of long-horizon and short-horizon tasks in a single view.

**Source**: BRD.md BR-004 (MEDIUM), PRD.md EPIC-003

## Scope
- Daily view: tasks with timeframe = DAILY
- Weekly view: tasks with timeframe = WEEKLY
- Monthly view: tasks with timeframe = MONTHLY
- Navigation between views
- Task count summary per horizon
