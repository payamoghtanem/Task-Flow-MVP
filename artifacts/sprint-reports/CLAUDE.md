# Sprint Reports Directory

## Role
Stores all periodic reports generated by agents: analysis reports, compliance reports, sprint review records, GDPR reports, security reports.

## Report Types
| Report | Produced By | Timing |
|--------|------------|--------|
| `analysis-sprint-N.md` | Product Analyst (04) | Sprint start |
| `compliance-report-sprint-N.md` | Traceability Agent (15) | Post-merge |
| `gdpr-compliance-report-sprint-N.md` | GDPR Agent (17) | Pre-deploy |
| `sprint-review-N.md` | Scrum Master (03) | Sprint end |
| `data-analysis-sprint-N.md` | Data Scientist (10) | On demand |

## Retention
Reports kept for 6 sprints then archived. Never deleted.

## Access
ALL agents read-only.
