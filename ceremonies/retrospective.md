# Retrospective — Ceremony Script v1.0

## Participants
Scrum Master (03) [facilitator], Lesson Learner (21), All agents

## Time Box
Max 1 hour. Sprint end.

## Agenda

### 1. Data Review (15 min)
Scrum Master (03) presents sprint metrics:
- Velocity trend
- Blocker count and resolution time
- Review cycle count (how many re-reviews per story)
- Test failure rate

### 2. Start / Stop / Continue (30 min)
Structured collection from all agents:
- **STOP**: What caused friction, rework, or delays?
- **START**: What should we try that we haven't done?
- **CONTINUE**: What worked well and must be preserved?

Each agent submits 1-3 items per category.

### 3. Action Item Selection (10 min)
- Scrum Master (03) facilitates voting on top 2-3 items
- Each action must have: owner agent, measurable outcome, sprint deadline

### 4. Lesson Learner Synthesis (5 min)
- Lesson Learner (21) confirms insights captured
- Standards update proposals identified

## Output
`ceremonies/retrospective.md` updated with sprint section.
Action items written to `product-management/sprints/current/tasks.md` as next-sprint items.
Lesson Learner (21) proceeds with Workflow 5 Steps 3-7.
