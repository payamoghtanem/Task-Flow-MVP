# Ceremonies Directory

## Role
Agile ceremony scripts and templates. Every ceremony produces a structured output document. Facilitating agent is Scrum Master (03) unless otherwise noted.

## Scope
- In: Sprint planning, daily standup, sprint review, retrospective, backlog grooming scripts
- Out: Code, architecture, testing, agent definitions

## Write Authority
Scrum Master (03) + Product Owner (02).

## Ceremony Index
- `backlog-grooming.md` — PO + Analyst ceremony
- `sprint-planning.md` — SM + PO planning ceremony
- `daily-standup.md` — Daily sync format
- `sprint-review.md` — Demo and acceptance
- `retrospective.md` — Continuous improvement ceremony

## Time Boxes
| Ceremony | Max Duration | Frequency |
|----------|-------------|-----------|
| Backlog Grooming | 2h | Weekly |
| Sprint Planning | 2h | Sprint start |
| Daily Standup | 15 min | Daily |
| Sprint Review | 1h | Sprint end |
| Retrospective | 1h | Sprint end |

## Constraints
- Time boxes are HARD limits — no ceremony runs over
- Output document must be produced before ceremony is marked complete
- Scrum Master (03) owns ceremony facilitation
