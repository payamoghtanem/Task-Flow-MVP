# Product Requirements Document (PRD)

## User Personas

1. **Alex (The Busy Professional)**
   - **Age/Demographic**: 34, Marketing Manager
   - **Pain Point**: Uses a heavy corporate tool for work, and sticky notes for home. Constantly drops the ball on personal errands.
   - **Need**: A single dashboard to see both contexts without mixing the actual tasks.

2. **Sarah (The Freelancer)**
   - **Age/Demographic**: 28, Graphic Designer (Non-technical)
   - **Pain Point**: Blurs the line between billable client work and personal administrative tasks, leading to burnout.
   - **Need**: A lightweight tool to quickly categorize tasks as "Professional" or "Personal".

3. **Jamie (The Graduate Student)**
   - **Age/Demographic**: 24, PhD Candidate
   - **Pain Point**: Overwhelmed by long-term thesis milestones vs. short-term daily homework/chores.
   - **Need**: A system that separates daily urgent items from monthly goals.

## Epics

## EPIC-001: User Onboarding
**User Persona**: Sarah (The Freelancer)
**Business Goal**: Reduce onboarding friction to <3 minutes
**Stories**: Registration, Profile Setup, Email Verification, Guided Tour.

## EPIC-002: Task Lifecycle Management
**User Persona**: Alex (The Busy Professional)
**Business Goal**: Allow seamless creation, modification, and deletion of tasks.
**Stories**: Create Task with metadata, Edit Task Details, Delete Task, Mark Task as Done.

## EPIC-003: Time-Horizon Views
**User Persona**: Jamie (The Graduate Student)
**Business Goal**: Improve planning and reduce cognitive load via timeframe separation.
**Stories**: Daily View Dashboard, Weekly View Dashboard, Monthly View Dashboard.

## EPIC-004: Context Switching & Filtering
**User Persona**: Alex (The Busy Professional)
**Business Goal**: Ensure users can focus on the right tasks at the right time.
**Stories**: Category Toggle (Personal vs. Professional), Visual Color-Coding for Categories.

## EPIC-005: Account & Security Settings
**User Persona**: Sarah (The Freelancer)
**Business Goal**: Maintain user trust and data privacy.
**Stories**: Change Password, Request Password Reset, Logout securely, Delete Account.
