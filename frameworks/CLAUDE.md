# Frameworks Directory

## Role
Methodological frameworks that govern how this team operates: Agile/Scrum execution rules, GDPR compliance checklist, and OWASP security constraints.

## Scope
- In: Scrum ceremonies definition, DoD criteria, GDPR compliance protocol, OWASP Top-10 enforcement rules
- Out: Code implementation, agent routing, sprint execution details

## Write Authority
Scrum Master (03) + Solution Architect (06). Updates require Lesson Learner (21) proposal.

## Frameworks Index
- `scrum/definition-of-done.md` — DoD v1.0
- `gdpr/gdpr-checklist.md` — GDPR compliance checklist v1.0
- `owasp/CLAUDE.md` — OWASP Top-10 agent constraint block v1.0

## Constraints
- DoD criteria are non-negotiable — stories not meeting DoD cannot close
- GDPR checklist must run before any deploy touching PII data
- OWASP constraints apply to all agents — no exceptions
